#
- umc 미션, 과제 수행 페이지
- 깃허브 사용법
  * 원격 저장소와 지역 저장소 연결
    git checkout <최종 개발 브랜치 이름>
    git pull origin <최종 개발 브랜치 이름>
    git fetch origin : 원격저장소를 최신화 해줌
    git checkout <issue브랜치 이름 or 새로 만든 브랜치>
    git pull origin <가져올 브랜치 이름>

  * 깃 브랜치 확인
    git branch

  * 깃 브랜치 선택/생성
    git checkout 브랜치이름

  * 깃 브랜치 삭제
    git branch -d 브랜치이름

  * 원격저장소의 깃 브랜치 삭제
    git push origin -d <원격 브랜치 이름>
